Camera Link Camera Management

Camera Link camera is the camera using Camera Link standard for data transmission. After adding Camera Link cameras to the Software, you can do further operations such as camera parameter configuration.

Note:

Camera Link is a serial communication protocol standard designed for camera interface applications based on the National Semiconductor interface Channel-link.
