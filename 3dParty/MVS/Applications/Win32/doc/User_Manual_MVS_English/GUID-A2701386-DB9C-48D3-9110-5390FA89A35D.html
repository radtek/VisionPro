File Access

The File Access feature allows you to export the User Set or DPC (Defective Pixel Correction) file of a connected device to the local PC as a binary file, or import a binary file from the local PC to a connected device.

Note:

  • The feature should be supported by the camera.

  • The File Access feature is available to use only when the camera is idle, i.e., not acquiring images.

  • For details about User Set, see User Set Control.
