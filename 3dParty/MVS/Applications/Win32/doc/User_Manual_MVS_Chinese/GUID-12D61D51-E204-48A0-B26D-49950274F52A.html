设置网格

客户端预览相机图像或本地图像时可设置网格。

客户端图像预览窗口已经在预览图像。
说明：

图像可通过相机实时预览获取，也可选择本地图像进行预览。

  1. 点击图像预览窗口右下角的，弹出设置窗口，如下图所示。
    图 1 设置网格
  2. 设置网格的间距，间距分为30、60、90和自定义。
    说明：

    选择“自定义”时，通过大小设置线宽。

  3. 点击颜色处的，进入颜色选择窗口，选择颜色并点击OK。
  4. 点击打开网格，如下图所示。
    图 2 设置网格
