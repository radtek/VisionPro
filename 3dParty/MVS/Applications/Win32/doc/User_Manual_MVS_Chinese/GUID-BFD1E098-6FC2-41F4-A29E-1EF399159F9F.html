客户端连接网口相机失败

问题描述

客户端能搜索到网口相机，但连接失败。

可能的原因

  • 原因1：网口相机与客户端不在同一个局域网内。

  • 原因2：网口相机已被其他程序连接。

解决方法

  • 针对原因1，使用IP配置工具修改IP地址，使相机与PC的IP地址处于同一个局域网。

  • 针对原因2，断开其他程序对相机的控制后，重新连接。
