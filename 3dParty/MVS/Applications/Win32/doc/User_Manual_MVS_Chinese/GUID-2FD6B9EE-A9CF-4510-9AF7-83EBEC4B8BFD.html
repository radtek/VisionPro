导出属性文件

客户端可通过文件存取的导出功能将相机属性以mfa格式文件导出。

客户端已连接相机。

  1. 点击属性区域的进入文件存取窗口。
  2. 选择需要导出的设备属性。
  3. 点击导出按钮。
  4. 选择导出文件的保存路径并保存。
    此时会显示实时导出进度直至提示“导出成功”。
