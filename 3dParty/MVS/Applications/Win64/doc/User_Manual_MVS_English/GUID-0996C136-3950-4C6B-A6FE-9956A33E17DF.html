Import User Set

You can import a binary file from the local PC to the User Set of the camera.

  1. Connect the camera to the Software.
  2. Click to open the File Access window.
  3. Select a User Set (User Set1, User Set2, or User Set3) or DPC from the drop-down list.
  4. Click Import to select the corresponding binary file and import it.
    Note:

    • DPC can only be imported to the same camera, while User Set can only be imported to the cameras of the same model.

    • The DPC will be imported and be effective directly. While for User Set, you should load the User Set to make it effective (see Step 5).

  5. If you select a User Set in step 3, load the User Set to make it effective.
    1. Click Feature Tree.
    2. Click to display the features under User Set Control.
    3. Select a User Set from User Set Selector.
    4. Click Execute to execute the User Set Load command to load the selected User Set.
