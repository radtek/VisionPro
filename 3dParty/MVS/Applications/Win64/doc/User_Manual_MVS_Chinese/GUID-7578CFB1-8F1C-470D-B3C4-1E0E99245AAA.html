导入/导出单个相机属性

客户端可通过属性区域进行导入/导出单个相机属性的操作。

客户端已连接相机。

导出功能较为简单，通过属性区域上方的操作即可。以下内容主要介绍如何导入单个相机属性。

  1. 点击属性区域上方的。
  2. 选择需要导入的mfs属性文件。
    说明：

    该功能只支持相同型号以及相同固件版本相机属性的导入。若型号或固件版本不匹配，则导入不成功。

  3. 若相机当前属性和mfs文件中的属性有所差别，点击打开后会显示具体的属性差异，如下图所示。
    图 1 属性差异查看
  4. 若确认无误则点击确认导入开始导入属性。若存在问题，则点击取消。
